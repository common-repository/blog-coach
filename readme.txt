Get Blogging!  Amp up your blogging with visual reminders in the admin toolbar to publish a new post. 

== Description ==
This plugin was designed to help you establish deadlines and keep your WordPress site fed with great new content. Most importantly, we hope you have a lot of fun.

Blogging can be both fun and rewarding, but it requires you to stay focus and dedicated to publishing amazing content. The Blog Coach plugin is a very simple plugin that provides powerful features to help you stay on track and encourage you to publish your next epic post. Most importantly, I hope this plugin helps you achieve more.

**Great Blogging Tips:**

Simple and non-obtrusive admin dashboard widget pulling great blogging focused content packed with tips and tricks for maximizing your blog and online marketing strategy.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/blog-coach` directory, or install the plugin from the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Use the new Blog Coach screen to configure the plugin
4. Select the desired reminders
